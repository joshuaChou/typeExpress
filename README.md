# typeExpress
Typescript practice  run server on docker

    Database:RethinkDB
    Queue:Rabbitmq
    Cache:redis
    Web:Express
    F2e:Angular2

Cleint and Server in one Project 

Build：

    npm build

Run client：

    npm start

Run Server:

    npm run server

Use PM2

    pm2 start processes.json
